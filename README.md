# Online Multiplayer Pixel Game

This is a turn-based online multiplayer pixel game where players throw stones and utilize power-up cards influenced by weather and physical conditions. Players have 10 turns to compete, and the winner is the one who deals more damage.

## Game Features

1. **Turn-Based Gameplay:** Each player takes turns throwing stones, 10 turns in total.
2. **Dynamic Factors:** 
   - Weather (e.g., rain reduces throw range, wind affects throw direction).
   - Player condition (e.g., fatigue, temperature impact accuracy).
3. **Power-Up Cards:** Cards grant special skills (e.g., fire stones, accuracy boost, damage block). Cards are drawn randomly to keep gameplay exciting.
